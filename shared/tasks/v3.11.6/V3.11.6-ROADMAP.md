# TaxFlow Pro v3.11.6 — Bookkeeping Platform + Hardening Roadmap

**Target branch:** `v3.11.6-dev` (cut from `v3.11.5` release tag)  
**Goal:** Deliver the full small-business bookkeeping platform expansion plus the hardening items deferred from v3.11.5.

---

## Release Bundles

| Bundle | Theme | Items |
|--------|-------|-------|
| **B1 — Foundation & Data Model** | PostgreSQL RLS, tenant boundary, Alembic migration to v3.11.6 schema, Chart of Accounts, Profile Roles | 2 items + 1 deferred hardening |
| **B2 — Core Transaction Engine** | Unified Register, recurring/scheduled transactions, check register, transaction splits | 3 must-have modules |
| **B3 — Assets, Liabilities & FX** | Loans/credit lines/investments, inventory & project tags, multi-currency | 3 must-have modules |
| **B4 — Financial Operations** | Bank reconciliation, tax filing exports, reports center, budget & cash flow forecasting | 4 must-have modules |
| **B5 — Invoicing & A/P / A/R** | Lightweight invoicing, accounts payable, accounts receivable | 1 must-have module |
| **B6 — Frontend UI Shell** | Unified register UI, COA tree, reports center, reconciliation, tax export, inventory, profile roles | 6 frontend tasks |
| **B7 — Packaging & Platform Hardening** | Single-instance enforcement, macOS `.app`/DMG, staged trust signals (OV/GPG/Apple) | 3 deferred hardening items |
| **B8 — Bank Parser Expansion** | Dedicated parsers for all remaining detected institutions (14 banks) | 14 parsers + unified test suite |

---

## Exit Criteria

- [ ] All 13 must-have bookkeeping modules implemented and tested.
- [ ] Unified register UI functional end-to-end (list, edit, split, delete transactions).
- [ ] PostgreSQL native RLS validated against a live PostgreSQL instance.
- [ ] Single-instance enforcement prevents stale server reuse on Windows and Ubuntu.
- [ ] macOS `.app` / DMG produced and smoke-tested.
- [ ] Full test suite passes (target: 0 failures).
- [ ] `CHANGES.md` Section 70+ documents all v3.11.6 work.
- [ ] Version bumped to `3.11.6` across all artifacts.
