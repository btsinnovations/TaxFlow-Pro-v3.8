# TaxFlow Pro v3.10 — Tooling & Security Hardening Spec

**Status:** Ready to scaffold; implementation starts after v3.9.2 security tasks complete  
**Branch:** `v3.10-dev` (cut from v3.9.2 release tag)  
**Goal:** Replace hand-built components with well-vetted offline libraries and close the remaining security gaps before v3.11 bookkeeping expansion.

---

## 1. Scope

v3.10 is a **non-feature release**. It does not add bookkeeping functionality. It makes the existing v3.9.2 pipeline:

- More secure
- More maintainable
- Better tested
- Easier to package

---

## 2. Guiding Principles

1. **Local-first / offline-only.** No cloud auth, no live feeds, no telemetry.
2. **Replace, then delete.** Every hand-built component gets a working replacement before the old code is removed.
3. **Security is not optional.** Every hardening task ships with tests.
4. **Validator impact matters.** Prefer changes that reduce custom code paths and increase test coverage.

## 5. Deferred Beyond v3.10

These are valid but too large or too speculative for this release:

- Memory-dump protection (S18)
- Installer/update channel integrity (S19)
- Privilege separation / separate worker user (S20)
- Physical device auto-lock integration (S21)
- Secure deletion of unencrypted temp files (S22)
- AI/LLM prompt injection (no LLM in use)
- Multi-currency, invoicing, A/P, A/R, payroll (v3.11)
- Electron/Tauri desktop wrapper (v3.11 packaging phase)

---

## 6. Deliverables

1. Prototype branch with at least **one end-to-end tooling replacement** (DuckDB export is recommended).
2. All P1 security tasks implemented and tested.
3. Updated dependency list with license review.
4. `CHANGES.md` Section 40+ documenting every replacement and hardening change.
5. Clear handoff doc to v3.11: which files were refactored, which patterns to follow.

---

## 7. Exit Criteria

- Full test suite passes.
- No P1 security task open.
- At least one tooling replacement fully integrated and tested.
- Packaging smoke test passes (installer builds, app starts, one PDF import works).
