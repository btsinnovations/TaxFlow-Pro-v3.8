## Recommended Architecture

### v3.10 Packaging Stack

```
PyInstaller (onedir)
├── Python runtime + site-packages
├── backend/ code
├── alembic/ migrations + alembic.ini
├── frontend/dist/ static assets
├── vendored/
│   ├── tesseract/        # tesseract.exe + tessdata
│   └── poppler/            # pdftotext.exe + pdfimages.exe + dlls
└── taxflow_launcher.exe
```

### Launcher flow

1. Determine `TAXFLOW_LOCAL_ROOT` (user-writable directory).
2. Ensure `db/`, `backups/`, `uploads/`, `ml/`, `logs/` subdirectories exist.
3. Set env vars for vendored binaries:
   - `TESSERACT_CMD`
   - `POPPLER_PATH`
4. Run Alembic `upgrade head` on the local DB.
5. Start Uvicorn with `backend.api:app` on `127.0.0.1:8000`.
6. Open browser to `http://localhost:8000`.
7. Block until Uvicorn process exits.

### Frontend serving

```python
from fastapi.staticfiles import StaticFiles

# Existing API routes registered under /api/*
app.mount("/", StaticFiles(directory="frontend/dist", html=True), name="static")
```

Order matters: API routes must be added **before** the static mount.
